## Match Types para Negativas

- **Broad match (negativa):** bloquea cualquier búsqueda que contenga ese término → máxima cobertura
- **Exact match (negativa):** solo bloquea esa query exacta → más preciso, menos cobertura
- **Phrase match (negativa):** bloquea la frase en ese orden

### Recomendación
- Términos conceptuales (gratis, empleo) → **broad match** negativa
- Términos específicos que NO quieres bloquear variaciones → **exact match** negativa
- Ejemplo: negativizar [fontanero gratis] en exact si SÍ quieres que aparezca "fontanero presupuesto gratis"
