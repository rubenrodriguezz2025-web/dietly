# Framework B2B SaaS — 4 Capas de Priorización (Adaptado España)

**Fuente:** Fibler (B2B SaaS Case Study), adaptado con perspectivas de Sylvia Perez y Aaron Young

## Estructura 4 Capas

| Capa | Tipo | Budget | Función | Match Types |
|---|---|---|---|---|
| **1** | High Intent | 58-80% | Captura fondo embudo | Exact + phrase |
| **2** | Competitor | ~15% | Robar market share | Exact + phrase |
| **3** | DSA | Bajo | Descubrimiento keywords | Dynamic |
| **4** | Retargeting | Bajo (€50/día) | Stay top of mind | YouTube/DG |

## Capa 1: High Intent (58-80% presupuesto)

- Keywords transaccionales: "[categoría] software", "[problema] plataforma", "[solución] herramienta"
- Hard exact + phrase match al inicio
- B2B España: CPCs €3-8 dependiendo del sector
- Landing pages dedicadas por grupo de keywords
- Conversión principal: demo request, free trial, contacto

## Capa 2: Competitor (15%)

- Keywords: "[competidor] alternativa", "[competidor] vs [tu marca]", "[competidor] precios"
- CPCs altos pero intención muy alta
- NECESITA landing pages de comparación dedicadas
- En España: competidores locales + internacionales
- No usar en campañas genéricas → campaña separada

## Capa 3: DSA (presupuesto bajo)

- Dynamic Search Ads apuntando a páginas producto que convierten
- Excluir: homepage, blog, careers, about, pricing (si no es LP de conversión)
- Función: minería de keywords → alimentar Capa 1
- Revisar query log semanalmente → mover winners a Capa 1

## Capa 4: Retargeting (€50/día)

- Remarketing visitantes Capas 1 y 2 en YouTube/Demand Gen
- Creativos: demos, case studies, testimoniales
- B2B: ciclos venta multi-touch → persistencia necesaria
- Presupuesto bajo pero impacto alto sobre audiencia caliente

## Setup B2B España

### Account Hygiene (anti-defaults)
1. Search Partners: OFF
2. Display Network: OFF
3. Crear campaña "sin objetivo" (evitar auto-broadening)
4. Desktop only para high ACV (mobile B2B = low-intent research)
5. Location: "Presencia" only

### Bidding B2B
- Fase 1: Manual CPC o Maximize Clicks con bid cap → hasta 15-20 conv/mes
- Fase 2: Maximize Conversions o tCPA con bid cap → volumen suficiente
- Fase 3: Mover conversión principal más abajo en funnel (de MQL a SQL a Closed Won)
- Brand: Target Impression Share con bid cap bajo (ahorra 40-50%)

### Negativas B2B (día 1)
1. Términos genéricos: gratis, curso, empleo, salario, reddit, pdf, qué es
2. Competidores (en campañas genéricas)
3. Marca propia (en campañas non-brand)
4. Exact + phrase match solo al inicio. Broad match quema presupuesto sin historial masivo

### OCI para B2B
- Esencial cuando escales más allá de search
- Feed back: Lead → MQL → SQL → Closed Won con valores incrementales
- MQL value = ~10% del valor medio cliente
- Usar deal CREATED date (no close date) para evitar ventana 90 días

## Métricas B2B España

| Métrica | Benchmark orientativo |
|---|---|
| CPC | €3-8 |
| CPL (form/demo) | €50-200 |
| Tasa cierre MQL → cliente | 5-15% |
| Ciclo venta | 30-90+ días |
| Presupuesto mínimo | €3.000-5.000/mes |
